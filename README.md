# 🌍 Breath Analyzer - Smart AQI Monitoring System

## 📌 Project Overview
Breath Analyzer is a smart air quality monitoring prototype that visualizes real-time PM2.5 pollution levels in Chennai using live API data and heatmap visualization.

## 🚀 Features
- Live AQI Monitoring Map
- PM2.5 Forecast Simulation
- Heatmap Pollution Visualization
- Auto Refresh Every 60 Seconds
- Risk Level Indicator (Good, Moderate, Unhealthy)

## 🛠 Technologies Used
- HTML
- JavaScript
- Leaflet.js (Map Visualization)
- OpenWeather API (Air Quality Data)

## ⚙ Setup Instructions
1. Download the project files
2. Open index.html in any browser
3. Internet connection required (for live API)
4. Map will auto-load Chennai AQI data

## 🎯 Prototype Goal
Proof of Concept for Smart Pollution Monitoring & Personalized Risk Analysis.
